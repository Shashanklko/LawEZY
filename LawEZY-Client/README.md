# 🖥️ LawEZY Client: The Institutional Portal

The LawEZY Client is the "face" of our mission to democratize legal access. Built with **React** and **Vite**, it provides a high-fidelity, accessible, and premium interface for both Citizens and Experts.

## 🎨 UI Philosophy: Accessibility & Trust
- **Glassmorphic Design:** A modern, transparent aesthetic that reflects our commitment to clarity and trust.
- **Responsive Layout:** Optimized for both desktop and mobile, ensuring users in any environment can access help.
- **Empathetic UX:** Simplified workflows to reduce the stress often associated with legal and financial matters.

## 🚀 Key Modules
- **LawinoAI Interface:** A clean, chat-focused environment for instant legal research.
- **Expert Nexus:** A directory and booking system to bridge the gap between people and professionals.
- **Community Hub:** A place for legal news and knowledge sharing.
- **Wallet & Identity:** Secure, transparent financial management.

## 🛠️ Development
```bash
npm install
npm run dev
```
